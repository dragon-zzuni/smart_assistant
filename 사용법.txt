Smart Assistant 사용법
=======================

🚀 빠른 시작
-----------
1. 더블클릭으로 실행: run_quick.bat
2. 터미널에서 실행: py quick_start.py

📱 기본 데모 (메신저 시뮬레이터)
---------------------------------
1. 더블클릭으로 실행: run_demo.bat  
2. 터미널에서 실행: py demo_simple.py

📧 이메일 포함 전체 테스트
--------------------------
1. 터미널에서 실행: py test_email.py
   (이메일 설정이 필요합니다)

🔧 문제 해결
-----------
- python 명령어가 안 될 때: py 사용
- 한글 깨짐: UTF-8 인코딩 확인
- 로깅 오류: 간단한 스크립트 사용 (quick_start.py)

📁 주요 파일
-----------
- quick_start.py: 가장 간단한 테스트
- demo_simple.py: 메신저 시뮬레이터 데모
- test_email.py: 이메일 포함 전체 테스트
- main.py: 메인 애플리케이션
- run_quick.bat: 빠른 시작 배치 파일
- run_demo.bat: 데모 실행 배치 파일

🎯 실행 순서 권장
---------------
1. run_quick.bat (가장 안전)
2. demo_simple.py (기능 확인)
3. test_email.py (이메일 테스트)
4. main.py (전체 시스템)

✅ 성공 확인
-----------
다음과 같은 출력이 나오면 성공:
- ✅ 5개 메신저 메시지 수집
- ✅ 5개 메시지 요약 완료
- ✅ 5개 메시지 분류 완료
- ✅ 6개 액션 추출 완료
- 📋 TODO 리스트 생성 완료

Smart Assistant GUI 사용법
========================

🚀 GUI 실행 방법
---------------
1. 더블클릭으로 실행: run_gui.bat
2. 터미널에서 실행: py run_gui.py

🖥️ GUI 주요 기능
---------------
1. 온라인/오프라인 상태 설정
   - 오프라인 → 온라인: 자동 모니터링 활성화
   - 온라인 → 오프라인: 수동 모드로 전환

2. 이메일 설정
   - 이메일 주소 입력 (예: imyongjun@naver.com)
   - 비밀번호/앱 비밀번호 입력
   - 제공자 선택 (naver, gmail, daum)

3. 메시지 수집
   - "메시지 수집 시작" 버튼 클릭
   - 진행률 표시 및 상태 메시지 확인

4. 결과 확인
   - TODO 리스트 탭: 추출된 액션들
   - 메시지 탭: 수집된 메시지 목록
   - 분석 결과 탭: AI 분석 결과

🧹 오프라인 정리 기능
------------------
1. "오프라인 정리" 버튼 클릭
2. 정리 설정 조정 (메시지 개수, 분석 옵션)
3. "정리 시작" 버튼으로 실행
4. 결과 확인 및 저장

📋 TODO 리스트 관리
-----------------
- 우선순위별 색상 표시 (🔴 High, 🟡 Medium, 🟢 Low)
- 요청자, 데드라인, 액션 타입 표시
- 상위 20개 아이템만 표시

🔧 exe 파일 생성
--------------
1. 더블클릭으로 빌드: build_simple.bat
2. 터미널에서 빌드: py build_simple.py
3. dist/SmartAssistant.exe 파일 생성

✅ 성공 확인
-----------
GUI가 정상적으로 실행되면:
- 좌측에 설정 패널 표시
- 우측에 탭 위젯 (TODO, 메시지, 분석 결과)
- 상태 표시기 (오프라인/온라인)
- 제어 버튼들 활성화

⚠️ 문제 해결
-----------
1. PyQt6 오류: conda install pyqt 실행
2. 한글 깨짐: UTF-8 인코딩 설정 확인
3. 이메일 연결 실패: IMAP 설정 및 앱 비밀번호 확인
4. exe 빌드 실패: PyInstaller 설치 확인

🎯 권장 사용 순서
---------------
1. run_gui.bat 실행
2. 이메일 설정 입력
3. 오프라인 상태에서 테스트
4. 온라인 상태로 전환하여 자동 모니터링
5. 필요시 오프라인 정리 기능 사용
